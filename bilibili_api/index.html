BiliBili接口整理

发表于 2018-03-20 | 阅读次数:

整理一下B站的接口（待补充）


所有接口都以AV20828860为例来解释说明

视频数据

https://api.bilibili.com/x/web-interface/archive/stat?aid=AV号
示例
https://api.bilibili.com/x/web-interface/archive/stat?aid=20828860
获取到如下数据

{
"code":0,    
"message":"0",
"ttl":1,
"data":
    {
  "aid":20828860,     //AV号
  "view":374364,    //播放量
  "danmaku":19705,    //弹幕数量
  "reply":4821,        //评论
  "favorite":15919,    //收藏
  "coin":29513,        //硬币
  "share":4360,        //分享
  "now_rank":0,        //当前排名
  "his_rank":17,    //历史排名
  "no_reprint":0,
  "copyright":1
    }
}

视频上方的数据

视频标签

https://api.bilibili.com/x/tag/archive/tags?aid=AV号
示例
https://api.bilibili.com/x/tag/archive/tags?aid=20828860

截图

视频评论

http://api.bilibili.com/x/v2/reply?type=1&oid=AV号
示例
http://api.bilibili.com/x/v2/reply?type=1&oid=20828860

搜索框默认展示词

https://www.bilibili.com/widget/getSearchDefaultWords
获取到如下内容
[{"show":"\u6296\u817f\u6311\u6218\uff01\u4f60\u80fd\u5fcd\u5f97\u4f4f\u5417","word":"av20980293"}]
Unicodez转码为中文
[{"show":"抖腿挑战！你能忍得住吗","word":"av20980293"}]
可以看出获得的是展示词以及其视频AV号
